getTier ( )

Returns the tier of the ChatInjection.
A tier classifies the type of the content (instead of the position as in type) of the injection. The return value should be one of:

  1. TIER_STATISTICS Optionally with the following bitmasks, combined with a bitmask OR (|):
  2. TIER_FLAG_TEXT (this bitmask is the implicit flag)
  3. TIER_FLAG_NUMERIC The tier will directly affect the position of the injection because the configuration file allows classifying and ordering injections according to their tiers.
    Returns
    int - the tier of the ChatInjection
